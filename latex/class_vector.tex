\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Vector$<$ T $>$}
Шаблонный класс динамической коллекции на основе динамического массива. 

Создать шаблонный класс динамической коллекции (связный список, очередь динамический массив). Обязательными пунктами задания являются следующие возможности коллекции\+:
\begin{DoxyItemize}
\item размещение в себе элементов произвольного типа;
\item добавление и удаление элементов;
\item предоставление доступа к элементам массива;
\item возможность работы с колеекцией через range based for, std\+::size;
\item корректная передача по параметру в функции. Дополнительные возможности\+: работа независимо с хранением данных и с выделением памяти; возможность удаления и вставки новых элементов в произвольные места коллекции. 
\end{DoxyItemize}
